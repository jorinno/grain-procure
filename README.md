# Grain Procurement System

A Python-based grain procurement project designed to calculate agent commissions and support structured transaction processing for grain-buying operations.

## Project Purpose

This project models part of a grain procurement workflow where field agents help process grain purchases and earn commission based on the weight of grain handled.

## Current Features

- Calculates commission based on grain category
- Supports different commission rates for maize and beans
- Separates business logic from direct input handling
- Prepared for future expansion into an API-based system
